# 🛍️ Customer Shopping Behavior Analysis

An end-to-end Data Analytics project that analyzes customer shopping behavior using **Python**, **MySQL**, and **Tableau**. The project focuses on cleaning raw data, performing SQL-based business analysis, and building an interactive dashboard to generate meaningful business insights.

---

## 📌 Project Overview

This project demonstrates a complete data analytics workflow starting from raw customer shopping data to an interactive business dashboard.

The analysis helps answer key business questions such as:

- Which customer segments generate the highest revenue?
- Do subscribed customers spend more than non-subscribers?
- Which product categories perform the best?
- What is the impact of discounts on customer purchases?
- Which age groups contribute the most revenue?
- Which products receive the highest customer ratings?

---

## 🎯 Business Objective

The objective of this project is to analyze customer purchasing patterns and provide actionable insights that can help businesses improve customer engagement, optimize product strategies, and increase overall revenue.

# 🧹 Data Cleaning

The following preprocessing steps were performed:

- Removed duplicate records
- Checked missing values
- Filled missing review ratings using median values
- Standardized column names
- Converted data types
- Created Age Group feature
- Created Purchase Frequency Days feature
- Exported cleaned dataset

# ⚙️ Feature Engineering

New features created:

# 📊 SQL Business Analysis

The following business questions were answered:

1. Total revenue generated by male vs female customers.
2. Customers who used discounts but spent above average.
3. Top 5 products with the highest average review rating.
4. Average purchase amount for Standard and Free Shipping.
5. Revenue comparison between subscribers and non-subscribers.
6. Products with the highest percentage of discounted purchases.
7. Customer segmentation into New, Returning, and Loyal.
8. Top 3 purchased products within each category.
9. Relationship between repeat buyers and subscription status.
10. Revenue contribution of each age group.

# 💡 Key Business Insights

- Clothing contributes the highest revenue among all product categories.
- Most customers are non-subscribers.
- Customers aged between 20–40 contribute the largest share of revenue.
- Customers using discounts still maintain a high average purchase amount.
- Accessories generate significant revenue despite fewer purchases.
- Customer review ratings remain consistently high across categories.
